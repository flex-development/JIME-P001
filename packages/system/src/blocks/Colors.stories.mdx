## Variables

File: `packages/system/src/scss/variables/_settings.scss`

```scss
$black: #000;
$red: #e40923;
$yellow: #fba422;
$white: #fff;

$dark: #28292c;
$light: #fcfcfc;
$primary: #012e65;
$secondary: #ce1127;
$ghost: transparent;

$danger: $secondary;
$disabled: #6c757d;
$muted: rgba($dark, 0.375);

$theme-colors: (
  'black': $black,
  danger: $danger,
  dark: $dark,
  disabled: $disabled,
  ghost: $ghost,
  light: $light,
  muted: $muted,
  primary: $primary,
  secondary: $secondary,
  'white': $white
);
```
